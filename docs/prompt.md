# Prompt de inicialización

Lee los archivos WORKFLOW.md y REQUIREMENTS.md antes de hacer nada más.

Si el proyecto tiene tests existentes, ejecútalos todos y reporta el resultado:
cuántos pasan, cuántos fallan, y el detalle de los que fallan.

Cuando tengas una visión clara del estado del proyecto, indícame que estás listo para recibir requerimientos.
